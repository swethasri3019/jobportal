#  Job Portal

This is a simple and user-friendly job portal website where users can search, filter, save, and apply for jobs easily.

##  Features

### 1.  Job Search & Filter
- Users can search jobs by job title easily  
- Filter jobs based on company and role  
- Helps to find suitable jobs quickly  
- Makes job searching simple and user-friendly  

### 2.  Job Details Display
- Shows salary, location, and role clearly  
- Displays multiple job opportunities in one place  
- Each job is shown in colorful and attractive cards  
- Helps users understand job information easily  

### 3.  Save & Apply Jobs
- Users can save their favorite jobs for later  
- Saved jobs can be removed anytime  
- Apply option is available for each job  
- Makes the portal more interactive and useful  

##  Technologies Used
- HTML  
- CSS  
- JavaScript  

##  Conclusion
This job portal simplifies the job searching process by providing an easy interface to explore, save, and apply for jobs in one place.
